# Agent Boundary

The old project mixed useful pipeline mechanics with AI-weekly-specific taxonomy. This new project keeps the agent layer empty until the PRD and backend architecture settle.

Planned agent responsibilities:

- collect public information from approved sources
- ingest manual notes, interviews, PDFs, and screenshots
- extract claims, metrics, risks, catalysts, and evidence links
- compare new evidence against active investment theses
- generate weekly briefs and decision memos with citations
- log pipeline decisions for auditability

Not planned for the agent:

- automatic trading
- unsourced recommendations
- hidden model decisions without an evidence trail
